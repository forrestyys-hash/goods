## 1. 프로젝트 개요

| 항목 | 내용 |
|:---|:---|
| **컨셉** | "The Electric Gallery" — 제품을 전시하는 프리미엄 갤러리형 쇼핑몰 |
| **타겟** | 프리미엄 굿즈·컬렉터블 상품을 구매하는 2030 세대 |

1. **시각적 차별화**: 기존 쇼핑몰과 차원이 다른 프리미엄 UX
2. **빠른 프로토타이핑**: Stitch로 디자인 → HTML 즉시 활용
3. **확장 가능한 구조**: Supabase 기반으로 실시간 데이터 연동 → 풀 기능 쇼핑몰로 성장

## 4. 단계별 개발 지침 ⚠️

> [!IMPORTANT]
> **핵심 규칙**: 각 Step이 완료되면 **반드시 사용자에게 완료 보고**를 하고,
> **승인을 받은 후에만** 다음 Step으로 진행합니다.
> 한 번에 여러 Step을 동시에 진행하지 않습니다.

### 개발 워크플로우

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ Step N   │ →  │ 개발     │ →  │ 완료     │ →  │ 승인     │
│ 착수     │    │ 진행     │    │ 보고 📋  │    │ 대기 ⏳  │
└──────────┘    └──────────┘    └──────────┘    └────┬─────┘
                                                     │
                                              승인 ✅ │
                                                     ▼
                                              ┌──────────┐
                                              │ Step N+1 │
                                              │ 착수     │
                                              └──────────┘
```

### 완료 보고 시 포함할 내용
1. ✅ 무엇을 완료했는지 (작업 내용)
2. 📁 생성/수정된 파일 목록
3. 🔍 확인이 필요한 사항
4. ➡️ 다음 Step 예정 내용
